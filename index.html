jstools

Bookmarklet: JSTools

Features

Add IDs

ids() will make sure everything in the DOM has an ID.

Compare styles

compare(a, b, ...) will return an object showing differences in the CSS of two or more elements. It will also outline the elements in the colours specified in compare.colours. By default it will not show border-width, for example, and include only border. Change compare.all to configure this.

There can be any number of arguments and they can be HTML elements, strings or numbers. If they're strings they'll be treated as element IDs, and if numbers they'll be converted to strings adn treated as element IDs too (this is really because ids() assigns numerical IDs and I'm lazy.

Key codes

Pressing any key will log the code to the console.
